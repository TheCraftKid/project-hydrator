# Project Hydrator
*A tool that stores project backups and provides a means of easily fetching them.*
For more information, see the [docs](https://williecubed.github.io/silver-pancake).

## Usage
### Installation

#### Node.js
Using NPM:
```bash
npm i -g spancake
```

Using Yarn:
```bash
yarn add spancake
```
